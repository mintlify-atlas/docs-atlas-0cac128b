## BasePlugin

The `BasePlugin` class is the base class for all angr management plugins. Plugins can extend the functionality of angr management by adding custom views, UI elements, analyses, and more.

### Constructor

```python
BasePlugin(workspace: Workspace)
```

<ParamField path="workspace" type="Workspace" required>
  The workspace instance this plugin is associated with
</ParamField>

### Class Attributes

<ResponseField name="DISPLAY_NAME" type="str | None">
  Human-readable name for the plugin. If None, uses the class name.
</ResponseField>

<ResponseField name="REQUIRE_WORKSPACE" type="bool" default="True">
  Whether this plugin requires a workspace to function
</ResponseField>

<ResponseField name="TOOLBAR_BUTTONS" type="list[tuple[QIcon, str]]" default="[]">
  List of toolbar buttons as (icon, tooltip) tuples
</ResponseField>

<ResponseField name="MENU_BUTTONS" type="list[str]" default="[]">
  List of menu button text labels
</ResponseField>

<ResponseField name="FUNC_COLUMNS" type="list[str]" default="[]">
  List of custom column names for the functions view
</ResponseField>

<ResponseField name="URL_ACTIONS" type="list[str]" default="[]">
  List of custom URL actions this plugin handles
</ResponseField>

<ResponseField name="CONFIG_ENTRIES" type="list[ConfigurationEntry]" default="[]">
  Custom configuration entries for this plugin
</ResponseField>

<ResponseField name="OPTIMIZATION_PASSES" type="list[tuple[type[OptimizationPass], bool]]" default="[]">
  Decompiler optimization passes as (pass_class, enabled) tuples
</ResponseField>

### Properties

<ResponseField name="workspace" type="Workspace | None">
  The workspace instance (may be None if REQUIRE_WORKSPACE is False)
</ResponseField>
