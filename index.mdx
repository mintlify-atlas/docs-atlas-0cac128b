  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with angr Management in minutes</p>

  <Steps>
    <Step title="Install angr Management">
      Install via pip or download a pre-built release:

      ```bash pip
      pip install angr-management
      ```

      Or download from [GitHub Releases](https://github.com/angr/angr-management/releases)
    </Step>

    <Step title="Launch the application">
      Run angr Management from your terminal:

      ```bash
      angr-management
      ```

      Or use the shorter alias:

      ```bash
      am
      ```
    </Step>

    <Step title="Load a binary">
      Open a binary for analysis using **File → Load Binary** or drag and drop a file into the window.

      <Note>
        angr Management supports multiple architectures including x86, x64, ARM, MIPS, and more.
      </Note>
    </Step>

    <Step title="Start analyzing">
      Explore the disassembly view, switch to the decompilation view, or use the interactive console to run custom angr analyses.
    </Step>
  </Steps>
</div>
